### অ্যারে ও অবজেক্ট 
```Python
//Array Declaration
ধরি ফলসমূহ = ["আম", "জাম", "কাঠাল","লিচু"];
// লিস্ট একসাথে প্রিন্ট হবে[]
লিখি(ফলসমূহ)  
// লিস্ট আলাদাভাবে প্রিন্ট হবে
লিখি(ফলসমূহ[0])
লিখি(ফলসমূহ[1])
লিখি(ফলসমূহ[2])
লিখি(ফলসমূহ[3])
```
