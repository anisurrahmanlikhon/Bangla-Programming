# বাংলা প্রোগ্রামিং ল্যাঙ্গুয়েজ ডকুমেন্টেশন

### ভেরিয়েবল ডিক্লেয়ারেশন

```Python
ধরি নাম = "আনিছুর";
ধরি বয়স = 29;
লিখি(নাম + " এর বয়স " + বয়স);
```
### শর্ত (কন্ডিশন স্টেটমেন্ট)
```Python
ধরি নাম = "আনিছুর";
যদি (নাম == "আনিছুর") {
লিখি("হ্যালো আনিছুর");
} নাহলে যদি (নাম == "লিখন") {
লিখি("হ্যালো লিখন");
} নাহলে {
লিখি("তোমার নাম মিলছে না");
}
```
### লুপ (লুপায়িত স্টেটমেন্ট)
```Python
// while লুপ
ধরি ক = 1;
ধরি খ = 10;
অব্দি(ক < খ) {
লিখি(ক);
ক = ক + 1;
}
```
```Python
// for লুপ
ধরি ক = 1;
ধরি খ = 10;
লুপ(ক = 0; ক < 10; ক++)
{
লিখি (ক);
}
```

ফল (অ + ব)
}

### বস্তু তৈরি করার কাজ
```Python
কাজ মানুষ(নাম, বয়স, বাসস্থান) {
 ফল { নাম, বয়স, বাসস্থান }
}

ধরি লিখন = মানুষ("লিখন", 22, "বাংলাদেশ");

লিখি (লিখন)
```
