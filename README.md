# বাংলা প্রোগ্রামিং ডকুমেন্টেশন

### ভেরিয়েবল

```
ধরি নাম = "আনিছুর";
ধরি বয়স = 29;
লিখি(নাম + " এর বয়স " + বয়স);
```
### শর্ত
```
ধরি নাম = "আনিছুর";
যদি (নাম == "আনিছুর") {
লিখি("হ্যালো আনিছুর");
} নাহলে যদি (নাম == "লিখন") {
লিখি("হ্যালো লিখন");
} নাহলে {
লিখি("তোমার নাম মিলছে না");
}
```
### লুপ
```
// while লুপ
ধরি ক = 1;
ধরি খ = 10;
অব্দি(ক < খ) {
লিখি(ক);
ক = ক + 1;
}
```
```
// for লুপ
ধরি ক = 1;
ধরি খ = 10;
লুপ(ক = 0; ক < 10; ক++)
{
লিখি (ক);
}
```

```

}

ধরি ফ = যোগ(5, 10);
```

### বস্তু তৈরি করার কাজ
```

```
